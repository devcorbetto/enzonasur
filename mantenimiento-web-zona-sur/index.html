Antes de comenzar

¿Trabajan cualquier tecnología?

Primero se revisa la plataforma, el acceso y el estado del proyecto para confirmar compatibilidad.

¿Incluye hosting?

Puede coordinarse, pero hosting y mantenimiento son conceptos diferentes y deben quedar claramente especificados.

¿Atienden urgencias?

La disponibilidad y los tiempos de respuesta dependen del plan contratado.

¿Hacen rediseños completos?

Los cambios mayores se evalúan como proyecto de diseño o desarrollo independiente.
